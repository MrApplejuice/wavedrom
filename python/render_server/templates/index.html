WaveDrom static image generator

Wavedrom is a tool programmed by Aliaksei Chapyzhenka that allows users to create technical timing diagrams for use in electronics. It comes with an online editor to assist the interactive construction of timing diagrams. However, the editor can only save the timing diagrams to local image files and to publish these images on forums or other open websites, one must first manually upload them to an image server because WaveDrom usually not supported natively.

This server offers Aliaksei's wavedrom-generator as a service. Paste your wavedrom source script below and copy the generated image link to embed the graphics into your text. Alternatively, you can use the REST-API to generate your images remotely.

REST API

The REST API can be used by anyone to remotely generate wavedrom plots that can be embedded in online media, solving the hosting issue for wavedrom users.

Link generator

REQUEST_TYPE: POST

URL: {{ protocol }}://{{ hostname }}/rest/generate_link?[type=png][scale=1.0]

The link generator receives text as input in the post body and compresses the received text using lzma compression. The maximum text length that can be submitted is 32768 bytes (truncated if necessary). The result will be a link returned as text with the embedded base64-encoded compressed text appended to the URL as "c"-parameter.

In order to obtain the rendered graphic just use the link returned by the call to generate_link.

Arguments

Image type (optional)

type=png|svg default: png

Determines the type of the image that should be generated.

Image scale (optional)

scale=[float] default: 1.0

Give the scale ofthe image that should be generated as a zoom factor. This value is only valid if type is png and will be ignored in all other cases. Minmal value is .1 and maximum value is 10.0. If the values lies outside the given range the result will be silently clipped to these maximum values.
